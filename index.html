Command List and Shortcuts

Commands:
/spawn (rarity) (mob) (count) - Spawns a mob.
/give (rarity) (petal) (count) - Gives a petal locally (token only).
/wave (count) - Sets the wave to the specified number.
/heart (count) - Gives you hearts.
/tp (count) - Gives you talent points.

Shortcuts:
W/A/S/D or Arrow Keys - Movement
Space - Attack (increases range)
Shift - Defend (decreases range)
Z - Toggle Inventory
C - Toggle Crafting / Trade near Trader
M - Toggle Big Map
X - Toggle Talents
G - Toggle Mob Gallery
V - Show Wave Rarity Chances
B - Show Mob Drop Chances
H - Show/Hide Hitboxes
O - Toggle God Mode
K - Show/Hide This Menu
N - Change Game Speed
L - Lure mobs
` - Developer Mode (Add Items)
I - Add Talent Points (Dev)
/ - Open Console
